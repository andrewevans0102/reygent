# Common Failures

This file documents recurring error patterns encountered during Reygent runs. Each entry includes:
- Description of the issue
- Number of occurrences and affected run IDs
- Last occurrence date
- Affected agent(s)
- Pattern/error signature
- Solution with code examples where applicable

Add entries using `reygent knowledge add-failure` or edit manually.
